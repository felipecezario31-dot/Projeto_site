(ENGAJAR PARA TRANSFORMAR)

TEMA DO PROJETO
Conexão Escola: Campanha de Engajamento Escolar com Uso de Indicadores e Plataforma Digital
1. INTRODUÇÃO
1.1 Problema
A escola enfrenta desafios relacionados à indisciplina, como atrasos frequentes, não utilização de uniforme, saídas de sala sem autorização, conflitos entre alunos e desmotivação em relação ao ambiente escolar.
Esses fatores impactam diretamente o processo de ensino-aprendizagem e o clima escolar.
1.2 Justificativa
O projeto surge da necessidade de promover maior engajamento dos alunos, incentivando o sentimento de pertencimento e responsabilidade.
A proposta está alinhada à formação do Técnico em Administração, ao desenvolver competências relacionadas à gestão, organização, análise de dados e tomada de decisão.
1.3 Objetivo Geral
Desenvolver uma campanha de engajamento escolar baseada em gestão e indicadores, com apoio de uma plataforma digital.
Objetivos Específicos
Identificar causas da desmotivação
Criar estratégias de engajamento
Desenvolver indicadores de acompanhamento
Aplicar ações no ambiente escolar
Avaliar resultados
2. DESENVOLVIMENTO
2.1 Descrição do Projeto
O projeto consiste na criação de uma campanha de engajamento escolar, estruturada a partir de três pilares: comunicação, gestão e monitoramento.
As equipes irão diagnosticar os principais problemas da escola, propor soluções práticas e acompanhar os resultados por meio de indicadores.
Como diferencial, será desenvolvida uma plataforma digital (site) para divulgação das ações, registro de dados e apresentação dos resultados.
2.2 Etapas
Diagnóstico do problema
Criação da campanha
Definição dos indicadores
Desenvolvimento do site
Aplicação das ações
Análise dos resultados
3. RECURSOS
Computadores
Formulários digitais
Materiais de divulgação
Plataforma digital
4. RESULTADOS ESPERADOS
Redução de atrasos
Maior uso de uniforme
Melhoria no comportamento
Aumento do engajamento
Participação ativa dos alunos
5. CONCLUSÃO
O projeto contribui para a formação de alunos mais conscientes, participativos e preparados para atuar de forma responsável na sociedade.
Além disso, apresenta uma solução viável e aplicável ao contexto escolar, promovendo melhorias reais no ambiente educacional.
